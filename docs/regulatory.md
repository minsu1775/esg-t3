## 1. 한국 — KSSB (한국회계기준원 지속가능성기준위원회)

### 1.1 KSDS (Korean Sustainability Disclosure Standards, 확정)

| 항목 | 내용 |
|---|---|
| 발표일 | 2026-02-26 |
| 표준 | KSSB 1 (일반 지속가능성) + KSSB 2 (기후 관련) |
| 정렬 | IFRS S1 / IFRS S2에 최대한 수렴, 한국 실정 반영 부속서(No. 101) 추가 |
| 필수 공시 | 기후(KSSB 2) 단독 의무화 — KSSB 1은 자발적 권고 출발 |
| Scope 3 | 의무 공시 시점 기준 3년 유예 → **사실상 2031년부터** |

### 1.2 금융위원회 공시 로드맵 (2026-02-25 초안, 4월 확정 예정)

> ⚠️ **2026-05-26 기준**: 4월 확정 여부 미확인. 다음 분기 검토 시 금융위원회 공식 발표 확인 후 갱신 필요.

| 의무화 시점 | 대상 | 최초 제출 연도 |
|---|---|---|
| FY 2028 | 코스피 상장 연결자산총액 **30조 원 이상** | 2029년 |
| FY 2029 | 코스피 상장 연결자산총액 **10조 원 이상** | 2030년 |
| FY 2030 | 코스피 상장 **전체** | 2031년 |
| FY 2031 | Scope 3 (3년 유예 적용) — 대형사부터 | 2032년 |

> ⚠️ **정책값 처리**: 위 일정은 확정 전이거나 의회 통과 절차 중임. 시스템에 하드코딩 금지. `disclosure_schedule` 메타데이터 테이블에서 관리. esg-t3에서도 Flyway V2 시드로 적재.

### 1.3 집권 여당 가속화 법안 (2026-03-14, 진행 중)

- Scope 3 의무화를 2031 → **2029년**으로 앞당기는 법안 발의.
- 확정 시 `disclosure_schedule` 업데이트로 흡수.

> ⚠️ **2026-05-26 기준**: 법안 현황 미확인 (국회 계류 중). 통과 시 본 문서 즉시 갱신 필요.

## 2. 국제 — ISSB (국제지속가능성기준위원회)

### 2.1 채택 현황 (2026-01 기준)

| 구분 | 국가 수 | 주요 국가 |
|---|---|---|
| 의무화 (2026 발효) | 6개 | 브라질, 멕시코, 칠레, 대만, 홍콩, 카타르 |
| 의무화 (예정) | 15개+ | 영국, 싱가포르, 캐나다, 일본, 호주 등 |
| 자발적 채택 | 다수 | 전 세계 GDP의 60% 이상 포괄 |

### 2.2 IFRS S1 / S2 핵심 요건

- **IFRS S1**: 일반 지속가능성 공시 (지배구조, 전략, 위험관리, 목표·지표)
- **IFRS S2**: 기후 관련 공시 (Scope 1·2·3, 기후 시나리오, 전환 계획)
- **TCFD 4대 기둥**: 지배구조 / 전략 / 위험관리 / 지표·목표 — IFRS S1/S2 구조와 일치

## 3. 유럽 — CSRD / ESRS

### 3.1 Omnibus I Directive (2026년 진행 중)

> ⚠️ **2026-05-26 기준**: EU 집행위원회 초안 제출됨, 유럽 의회 협상 진행 중. 최종 확정 전 본 문서 내용은 초안 기준임.

| 항목 | 변경 내용 |
|---|---|
| 적용 일정 | 수정안 목표: FY 2027부터 (조기 적용 2026 가능) |
| 의무 데이터포인트 | **60% 이상 감소** (전체 기준 70% 이상) |
| 중소기업(SME) | Wave 3 (상장 중소기업) 의무 대상 제외, **VSME** 자발적 표준으로 |
| 가치사슬 요청 | 직원 1,000명 이하 기업: 의무 표준 이하 데이터 제공 거부 가능 |

### 3.2 이중 중대성 (Double Materiality Assessment, DMA)

- CSRD의 핵심 원칙 유지: **재무 중대성** + **영향 중대성** 동시 평가
- AI 도구로 80% 이상 자동화 가능 (2026 시장 표준화)
- esg-t3는 MVP 범위 외 (M+1) — design.md §1.3 참조

## 4. GHG Protocol 개정 동향

### 4.1 Scope 3 Standard Phase 1 Update (2026-03 발표)

| 변경 항목 | 내용 |
|---|---|
| 95% 포함 임계값 | 의무 Scope 3 출처의 95% 수량화 후 제외 가능, 모든 제외 이유 공개 |
| Category 16 신설 | 촉진 활동(Facilitated Emissions) — 제3자 활동을 기업이 가능하게 하는 배출 |
| 데이터 품질 | 지출 기반 → 공급업체 특정 데이터로 전환 투명성 요구 |
| 최종 확정 | 공개 협의 초안 2026년 중반 예정, 최종 기준 **2027년 말** 예상 |

### 4.2 esg-t3 대응 전략

- **Scope 3는 MVP Out → M+1** (design.md §1.3, ADR-013)
- 단, 데이터 모델에 `scope` / `supplier_id` / `data_quality` 컬럼 사전 포함 (design.md §7.3) — M+1 활성화 비용 0
- Category 16(Facilitated Emissions) — 최종 기준 확정 후 검토
- 95% 포함 임계값 — `scope3_coverage_report` 자동 생성 기능은 M+1에서 추가

> 참고: esg-t2는 Scope 3 Cat 1·2·11을 MVP에 포함했으나, esg-t3는 ChatGPT 권고 수용 + 운영·거버넌스 강화 집중을 위해 양보. 도메인 모듈 슬롯은 동일하게 준비.
